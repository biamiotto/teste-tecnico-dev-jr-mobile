# 🌸 Perfil Dev Jr

## 📖 Descrição

Aplicativo desenvolvido em React Native com Expo como parte de um teste técnico para a vaga de Desenvolvedor Mobile Júnior.

O objetivo do projeto é apresentar um perfil profissional por meio de uma interface simples, organizada e intuitiva, utilizando navegação por abas inferiores. O aplicativo contém informações pessoais, habilidades técnicas, projetos desenvolvidos e uma área de contato.

Além da implementação dos requisitos propostos, foi aplicada uma identidade visual em tons de rosa bebê para proporcionar uma experiência mais agradável e moderna.

---

* Expo

* Navegação por abas inferiores (Bottom Tabs)

* Exibição de imagem de perfil
* Lista de habilidades

## 📱 Telas

### 🌸 Perfil

A tela de perfil apresenta informações pessoais do desenvolvedor, incluindo:

* Foto de perfil
* Nome
* Cargo desejado
* Seção "Sobre Mim"

---

### 💖 Habilidades

Tela destinada à apresentação das principais habilidades técnicas:

* React Native
* React
* JavaScript
* HTML
* CSS

As habilidades são exibidas em formato de lista para facilitar a leitura.

---

### 📂 Projetos e Contato

Tela com informações sobre projetos realizados e uma área de contato contendo:

* E-mail
* GitHub
* LinkedIn
* Campo para mensagem
* Botão "Enviar Contato"

## 🎨 Decisões de Desenvolvimento

O projeto foi desenvolvido utilizando componentes básicos do React Native para demonstrar domínio dos conceitos fundamentais exigidos para a vaga.

Foi utilizada a biblioteca React Navigation para implementação da navegação por abas inferiores, mantendo o código organizado e escalável.

A estilização foi realizada exclusivamente com StyleSheet, conforme solicitado na atividade.

## ▶️ Como Executar o Projeto

### Clone o repositório

```bash
git clone https://github.com/biamiotto/teste-tecnico-dev-jr-mobile.git
```

### Acesse a pasta do projeto

```bash
cd perfil-dev-jr
```

### Instale as dependências

```bash
npm install
```

### Execute o projeto

```bash
npx expo start
```
